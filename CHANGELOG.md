# Changelog

All notable changes to Served.SDK will be documented here.

## [2026.2.1] - 2026-02-01

### Added
- Public documentation repository (docs-only)
- NON-AI-MIT license to protect against AI training
- Comprehensive API reference for Projects, Tasks, TimeRegistration
- Getting started and authentication guides
- Usage examples and patterns

### Changed
- Source code moved to private Forge repository
- Documentation separated from source for public access

### Security
- Added robots.txt blocking AI crawlers (GPTBot, CCBot, etc.)

## [2026.1.x] - Previous Releases

See [NuGet package history](https://www.nuget.org/packages/Served.SDK) for previous versions.

## Versioning

We use [CalVer](https://calver.org/) with format `YYYY.MINOR.PATCH`:
- `YYYY` - Year of release
- `MINOR` - Feature release number within year
- `PATCH` - Bug fix release
